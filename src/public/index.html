API carrito

Para realizar consultas a la API de productos se debe realizar:

Ruta: localhost:8080/api/products

la misma utiliza los metodos GET, POST, PUT y DELETE

realizando solo GET trae todos los productos, con localhost:8080/api/products/(n) siendo n el numero de id, trae solo ese producto

PUT con localhost:8080/api/products/(n) siendo n el numero de id, se modicifa el producto elegido

DELETE con localhost:8080/api/products/(n) siendo n el numero de id, elimina solo ese producto


Para realizar consultas a la API de carrito se debe realizar:

Ruta: localhost:8080/api/carts

la misma utiliza los metodos GET, POST y DELETE

realizando solo GET trae todos los carritos, con localhost:8080/api/carts/(n) siendo n el numero de id, trae solo ese carrito

realizando solo POST trae genera un nuevo carrito

realizando POST con localhost:8080/api/carts/(n)/product/(pd) siendo n el numero de id de carrito y pd el id de producto que se va a cargar

DELETE con localhost:8080/api/carts/(n) siendo n el numero de id, elimina solo ese carrito
